# AI Career Advisor Chatbot

## Project Overview
AI-powered Career Advisor Chatbot built using:
- Python
- Streamlit
- Gemini GenAI API
- Prompt Engineering

The chatbot provides:
- Career guidance
- AI/ML roadmap
- Full stack roadmap
- Resume tips
- Interview preparation guidance

---

## Features
- Multi-turn conversation
- Interactive Streamlit UI
- Gemini API Integration
- Modular architecture
- Prompt engineering
- Logging support
- Environment variable security

---

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- dotenv
